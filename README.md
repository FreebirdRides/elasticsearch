# Elasticsearch, Logstash, Kibana (ELK) Docker image

### Freebird specific
* Elasticsearch ONLY
* When doing a fresh deploy, you'll need to scale after the app has been deployed `convox scale elasticsearch --memory=2048 --cpu=2048`
